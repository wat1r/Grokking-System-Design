## 单位换算表

```java
1B  = 8 bit         byte
1KB = 1024bytes     Kilobyte  
1MB = 1024KB        Megabyte   //百万字节 兆
1GB = 1024MB        Gigabyte   //亿字节，又称“千兆”
1TB = 1024GB        Terabyte   //万亿字节，太字节
1PB = 1024TB        Petabyte   // 千万亿字节，拍字节
1EB  =1024PB        Exabyte    //百亿亿字节，艾字节
...
1GB = 10^8 bytes   
```
